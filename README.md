# 🏦 Bank Management System (Java)

## 📌 Description
A console-based banking application built using Java.  
It allows users to perform basic banking operations securely.

## 🚀 Features
- Withdraw money (PIN protected)
- Deposit money
- Check account balance
- Update user details
- Menu-driven interface

## 🛠 Technologies Used
- Java
- OOP (Object-Oriented Programming)
- Scanner Class

## ▶️ How to Run
1. Compile:
   javac Bank_Management_system.java User_class.java

2. Run:
   java User_class
